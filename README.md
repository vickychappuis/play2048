# Play2048

A little brain snack to play while your agents finish their work.

Built with [Ink](https://github.com/vadimdemedes/ink) and Node.js. Runs in your terminal.

Quick play
```bash
# run without installing
npx play2048

# or install globally to get a persistent `play2048` command
npm install -g play2048
play2048
```

Controls
- Arrow keys or WASD — move tiles
- `r` — restart
- `q` or `esc` — quit
- `ctrl+c` — quit
